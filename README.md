## 🧑‍💻 What I Work On

| Area | What I Do |
|------|-----------|
| **RTL Design** | Synthesisable modules in Verilog/SV — ALUs, FIFOs, FSMs, pipelines |
| **Testbench** | Directed & constrained-random testbenches in SystemVerilog |
| **Coverage** | Covergroups, coverpoints & cross-coverage for closure |
| **Assertions** | SVA `assert`, `assume`, `cover` properties for formal & simulation |
| **Synthesis** | RTL-to-gate with Cadence Genus, SDC constraints, timing analysis |
| **HDLBits** | Daily Verilog problems — combinational, sequential, FSMs |
