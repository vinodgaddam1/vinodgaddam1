## 🧑‍💻 What I Work On

| Area | What I Do |
|---|---|
| **RTL Design** | Write synthesisable modules in Verilog/SV — ALUs, FIFOs, FSMs, pipelines |
| **Testbench** | Build directed & constrained-random testbenches in SystemVerilog |
| **Coverage** | Define covergroups, coverpoints & cross-coverage for closure |
| **Assertions** | Write SVA `assert`, `assume`, `cover` properties for formal & simulation |
| **Synthesis** | Run logic synthesis with Cadence Genus, analyse timing paths |
| **HDLBits** | Solve digital design problems daily — combinational, sequential, FSMs |
